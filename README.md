# Hey, I'm Kami

I'm mostly a game developer, but I also like building small interactive things with code.

Most of my work is focused on games, tools, prototypes, and little web experiments. Some of the projects here are JavaScript browser games that use public APIs, like country data or Pokémon data. I like making things that are simple to play with, but still teach me something while building them.

GitHub is where I host some of my code and web projects. Some of the browser projects can be played directly through GitHub Pages, and I also collect my projects and write about the process on my blog.
